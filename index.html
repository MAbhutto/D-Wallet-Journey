About the Project

D-Wallet Journey aims to delve deep into the realm of digital wallets, focusing on understanding user personas and multi-service usage patterns. By analyzing user behaviors related to transfers, bill payments, mobile packages, and mobile recharges, we aim to uncover the intricate patterns and preferences that shape user experiences.

Features

  • Advanced analytics including Pareto analysis
  • Multi-service usage patterns exploration
  • User personas analysis
  • Interactive dashboards and visualizations
